# 🥧 PiFlow - 円周率記憶トレーニングツール

円周率（π）の小数点以下の桁を効率的に記憶・練習するためのWebアプリケーション

## ✨ 特徴

- 🎯 **リアルタイムフィードバック**：入力した瞬間に正誤がわかる
- 📊 **詳細な統計**：進捗グラフ、弱点分析、練習履歴を可視化
- 🎨 **洗練されたUI**：ミニマルで集中できるデザイン
- 📱 **完全レスポンシブ**：スマホ・タブレット・PCに対応
- 💾 **完全ローカル**：データはすべてブラウザ内に保存、プライバシー保護
- ⚡ **超高速**：Vite + Reactで瞬時に起動

## 🎮 モード

1. **フリーモード**：エラーまで何桁記憶できるかチャレンジ
2. **タイムアタックモード**：制限時間内に何桁入力できるか
3. **復習モード**：過去に間違えた範囲を集中的に練習

## 📖 使い方（実装後）

1. **練習開始**
   - ページを開くと自動的に入力欄にフォーカス
   - `Enter`キーまたは数字を入力して開始

2. **数字を入力**
   - 円周率の小数点以下を1桁ずつ入力
   - 正解：緑色のフィードバック → 次の桁へ
   - 不正解：赤色のフィードバック → 結果表示

3. **統計を確認**
   - ヘッダーの「統計」ボタンをクリック
   - パーソナルベスト、弱点分析、進捗グラフを確認
   - 弱点の「復習」ボタンで該当範囲を練習

4. **キーボードショートカット**
   - `Enter`：練習開始 / リトライ
   - `Space`：一時停止
   - `Esc`：練習中断
   - `s`：統計画面へ

## 🎨 デザインコンセプト

### ミニマリズム
余計な要素を排除し、練習に集中できるシンプルな画面設計

### 即時フィードバック
すべての操作に瞬時に反応し、ストレスフリーな体験を提供

微細だが効果的なモーションで、ユーザーの行動をサポート
