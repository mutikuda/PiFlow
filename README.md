# π PiFlow - 円周率記憶トレーニングツール

円周率（π）の小数点以下の桁を効率的に記憶・練習するための、数学的で美しいWebアプリケーション

**MASTER THE INFINITE**

## ✨ 特徴

- 🎯 **リアルタイムフィードバック**：入力した瞬間に正誤がわかる
- 📚 **プラクティスモード**：間違えた後も学習を継続、正解すればプレイモードに復帰
- 🎨 **数学的デザイン**：幾何学的なグリッドパターンとブルー/シアングラデーション
- 📱 **完全レスポンシブ**：スマホ・タブレット・PCに対応、画面内に常に収まるレイアウト
- 💾 **完全ローカル**：データはすべてブラウザ内に保存、プライバシー保護
- ⚡ **超高速**：Vite + Reactで瞬時に起動
- ⌨️ **デュアル入力**：キーボードとタッチパッドの両方に対応

## 🎮 ゲームフロー

### 1. アイドル状態
- **「練習開始」** ボタンまたは **Enter** キーでゲーム開始
- あなたのベスト記録が表示される

### 2. プレイング（Playing）モード
- `3.` の後に続く円周率の小数点以下を1桁ずつ入力
- 正解：緑色のフィードバック + 次の桁へ
- 不正解：プラクティスモードへ移行

### 3. プラクティス（Practice）モード 📚
- 間違えた位置と到達桁数を表示
- 次の10桁を参考として表示（グレーアウト）
- **入力を継続可能** - 正解を入力すればプレイングモードに復帰
- 何度でも挑戦できる学習モード

### 4. ゲームリセット
- 「もう一度チャレンジ」ボタンでアイドル状態に戻る

## 🎯 使い方

### 基本操作

1. **練習開始**
   - `Enter`キーまたは「練習開始」ボタンをクリック

2. **数字を入力**
   - **キーボード**: `0-9`キーを押す
   - **タッチパッド**: 画面下部のNumPadをタップ
   - 円周率 `3.` の後に続く数字を1桁ずつ入力

3. **フィードバック**
   - ✓ 正解：緑色のグロー + 次の桁へ
   - ✗ 不正解：赤色のシェイク + プラクティスモードへ

4. **プラクティスモード**
   - 次の10桁を参照しながら学習
   - 正解を入力して継続
   - または「もう一度チャレンジ」で最初から

### キーボードショートカット

- `Enter`：練習開始
- `0-9`：数字入力
- マウス/タッチ：NumPadクリック/タップ

## 📊 記録管理

すべてのデータはブラウザのLocalStorageに保存されます：

- 最高桁数（maxDigits）
- 記録達成日時（maxDigitsDate）
- 総セッション数（totalSessions）
- 総入力桁数（totalDigitsTyped）
